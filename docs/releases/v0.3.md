# XLang v0.3 Release Notes

**Status:** Implemented in bootstrap compiler  
**Builds on:** [v0.2](v0.2.md)

v0.3 adds stack-allocated struct values with scalar fields, per [RFC-0018](../rfcs/RFC-0018-v0-3-roadmap-and-scope.md) through [RFC-0022](../rfcs/RFC-0022-llvm-struct-lowering.md).

---

## New language features

| Feature | RFC |
|---------|-----|
| Struct layout (`i32` / `bool` fields) | [RFC-0019](../rfcs/RFC-0019-struct-layout-and-declarations.md) |
| Struct locals + literals | [RFC-0020](../rfcs/RFC-0020-struct-literals-and-construction.md) |
| Field read `p.x` | [RFC-0021](../rfcs/RFC-0021-field-access-and-assignment.md) |
| Field assign `p.x = expr;` | [RFC-0021](../rfcs/RFC-0021-field-access-and-assignment.md) |
| LLVM struct lowering | [RFC-0022](../rfcs/RFC-0022-llvm-struct-lowering.md) |

### Example

```xlang
struct Vec2 {
    i32 x;
    i32 y;
}

i32 main() {
    Vec2 p = { 3, 4 };
    return p.x + p.y;  // 7
}
```

---

## Still not included

Struct parameters/returns, nested struct fields, struct arrays, `str` struct fields at codegen, methods, generics.

---

## Quick start

```bash
cargo run --manifest-path compiler/Cargo.toml -- run examples/v0.3/main.x
cargo test --manifest-path compiler/Cargo.toml
```

Expected exit code: **7**.
