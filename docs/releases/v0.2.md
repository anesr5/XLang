# XLang v0.2 Release Notes

**Status:** Implemented in bootstrap compiler  
**Builds on:** [v0.1](v0.1.md)

v0.2 adds iteration and fixed-size stack arrays with runtime bounds checking, per [RFC-0014](../rfcs/RFC-0014-v0-2-roadmap-and-scope.md) through [RFC-0017](../rfcs/RFC-0017-index-expressions-and-bounds-checking.md).

---

## New language features

| Feature | RFC |
|---------|-----|
| `while` loops | [RFC-0015](../rfcs/RFC-0015-while-loops-break-and-continue.md) |
| `break` / `continue` | [RFC-0015](../rfcs/RFC-0015-while-loops-break-and-continue.md) |
| Fixed-size arrays `T[N]` | [RFC-0016](../rfcs/RFC-0016-fixed-size-arrays.md) |
| Array literals `{ a, b, … }` | [RFC-0016](../rfcs/RFC-0016-fixed-size-arrays.md) |
| Index expressions `arr[i]` | [RFC-0017](../rfcs/RFC-0017-index-expressions-and-bounds-checking.md) |
| Runtime bounds checks → `llvm.trap` | [RFC-0017](../rfcs/RFC-0017-index-expressions-and-bounds-checking.md) |

### Example

```xlang
i32 main() {
    i32[4] xs = { 1, 2, 3, 4 };
    i32 total = 0;
    i32 i = 0;
    while i < 4 {
        total = total + xs[i];
        i = i + 1;
    }
    return total;  // 10
}
```

---

## Syntax summary

| Construct | Form |
|-----------|------|
| Array local | `i32[N] name = { … };` |
| Immutable array | `const i32[N] name = { … };` |
| Index read | `xs[i]` |
| Index assign | `xs[i] = expr;` |
| While | `while cond { … }` |
| Break | `break;` (inside `while`) |
| Continue | `continue;` (inside `while`) |

- Array length `N` must be a **positive integer literal** (`N >= 1`, max 65535).
- Array literal must have **exactly `N` elements**.
- Index must be **`i32`**. Constant out-of-bounds indices are rejected at compile time.
- Non-constant indices emit **runtime bounds checks**.

---

## Still not included

`for` loops, dynamic arrays, heap allocation, pointers, ownership, struct values, generics, traits, GPU, async, package manager, `str` codegen.
